My Projects

1) FACE RECOGNITION SYSTEM

In this project I have used Local Binary Patterns Histograms (LBPH) algorithm to train the classifiers.

How This Code Works?

  • Person Details: This function is designed to collect and store individuals' information in a database. Additionally, it captures the person's image for training purposes.
  • Train Data: Clicking this button opens a new window where data is trained using the Local Binary Patterns Histograms method. The trained data is then stored in the classifiers.xml file.
  • Photos: By clicking this button, all the photos taken are displayed.
  • Scan Face: Upon clicking this button, a new window opens for real-time scanning of a person's face. The details of the scanned face are displayed in real-time.

2) AI ASSISTANT FOR PC

In this project, I utilized the Python language to create an AI ASSISTANT FOR PC, aiming to assist with various day-to-day tasks.

Project Features:

  • Functionality: The AI assistant can perform a range of tasks such as opening applications, telling jokes, and reading news.
  • API Integration: API keys have been incorporated to enable news reading, and additional API keys can be added to enhance functionality.
  • Application Control: The assistant is capable of opening applications like Word, PowerPoint, Google, etc.
  • Web Browsing: Users can instruct the assistant to open commonly used websites or specify a particular website.
  • Mathematical Operations: Basic mathematical operations can be performed using the assistant.
